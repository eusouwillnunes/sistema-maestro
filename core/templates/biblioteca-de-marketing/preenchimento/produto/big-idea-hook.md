# Big Idea & Hook

> [!info] Guia de preenchimento
> A Big Idea e Hook são o conceito central da comunicação de vendas. Usado pelo Copywriter e Estrategista para criar campanhas com mensagem unificada.
> Camada 3: enriquece a estratégia criativa mas não bloqueia.
> Preencha cada seção ou peça ao Maestro para ajudar.

## Características de uma Big Idea Forte

- [ ] É **nova** ou apresentada de forma nova
- [ ] É **única** — só você pode dizer isso
- [ ] É **simples** — fácil de entender
- [ ] É **emocional** — provoca reação
- [ ] É **memorável** — gruda na mente

## O Hook (Gancho)

> Você tem 3 segundos para capturar atenção.

| Tipo | Descrição | Exemplo |
|------|-----------|---------|
| **Pergunta** | Faz pensar | "E se você pudesse...?" |
| **Declaração chocante** | Quebra padrões | "Tudo que você aprendeu está errado" |
| **História** | Conecta emocionalmente | "Há 3 anos, eu estava..." |
| **Promessa** | Resultado específico | "Como conseguir X em Y dias" |
| **Prova** | Credibilidade imediata | "127 profissionais já..." |
