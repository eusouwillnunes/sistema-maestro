# Swipe File

> [!info] Guia de preenchimento
> O Swipe File é seu banco de referências de copy e criativos do nicho. Usado pelo Copywriter como inspiração e calibração.
> Camada 3: enriquece a criatividade mas não bloqueia.

## Como Usar

> Não copie literalmente. **Analise** o que funciona, **entenda** o princípio e **adapte** para seu contexto.
