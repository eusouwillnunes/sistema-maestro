# Feedback do Revisor

> Registro contínuo de divergências entre o Revisor e o usuário. Processado periodicamente via `/maestro-revisar-memorias` pra ajustar a calibragem do projeto.
>
> Toda entrada vai pra uma das 3 seções abaixo. Quando processado em revisão de memórias, a entrada migra pra "Processado" com linha de resolução.

## Falsos negativos
> Casos onde o Revisor aprovou texto que tinha padrão de IA.
